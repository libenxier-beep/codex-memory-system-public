# Storage And Routing

- `memory-sidecar/`: raw or near-raw evidence kept for backtracking
- `memories/rollout_summaries/`: complete round retrospectives and promotion candidates
- `memories/core|platform|learnings/`: durable work memory after conservative filtering
- `personal_memory/`: durable private growth, value, and pattern material

Prefer the least durable layer that still preserves the useful signal.
