# Commit Contract

## Inputs

- completed Distillation Report
- canonical candidate list
- reject-only user feedback
- current work-memory and personal-memory truth-source files

## Required candidate fields

- `candidate_id`
- `summary`
- `why`
- `scope_hint`
- `recommended_destination`
- `draft_write`

## Outputs

- Commit Report
- minimal file updates in the selected memory layer

## Non-goals

- inventing new candidates
- redefining memory boundaries inside the skill
